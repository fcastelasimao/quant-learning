## Module Map

| File | Purpose |
|---|---|
| `config.py` | All parameters. Loads allocation from `strategies.json` |

| `backtest.py` | Simulation engine: `run_backtest`, `run_backtest_rolling_rp`, `compute_stats` |
| `optimiser.py` | `compute_risk_parity_weights()` (SLSQP), random/calmar search |
| `data.py` | yfinance price fetching with data quality checks |
| `export.py` | Results folders, Excel master log, terminal formatting |
| `plotting.py` | Dark-theme matplotlib charts |
| `portfolio.py` | Live portfolio state and rebalancing instructions |
| `validation.py` | Walk-forward and Pareto frontier analysis |
| `compare_allw.py` | ALLW ETF head-to-head comparison |

## Key Constraints

- **IS/OOS discipline:** Never optimise on data after OOS_START. RP covariance uses `end_date` parameter to enforce this.
- **Calmar ratio** is the primary evaluation metric (CAGR / |max drawdown|).
- **Risk parity** does not optimise for returns — it equalises risk contributions via covariance matrix only.
- Production weights: SPY 13%, QQQ 11%, TLT 19%, TIP 33%, GLD 14%, GSG 10%.
- `results/` is .gitignore'd — all output is regenerated by running scripts.
